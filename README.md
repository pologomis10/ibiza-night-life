INFORME IBIZA NIGHT LIFE

Descripció del projecte
Aquesta pàgina web que he elaborat tracta sobre la vida nocturna a Ibiza, la qual està anomenada “Ibiza Night Life”. El seu objectiu principal és mostrar informació sobre les principals discoteques, festivals i zones d’oci mitjançant una web moderna, visual, interactiva i responsive.
Està formada per diferents apartats:
Inici.
Informació sobre Ibiza.
Multimèdia.
Mapa interactiu.
Formulari de contacte.

Tecnologies utilitzades
Per a desenvolupar aquest projecte, he utilitzat les següents tecnologies:
HTML per a estructurar el contingut de la web.
CSS per al disseny visual i els estils personalitzats.
JavaScript per al funcionament del mapa interactiu des de QGIS.
Bootstrap com a framework CSS per adaptar la pàgina a diferents dispositius i facilitar el disseny responsive.



Mapa interactiu
El mapa interactiu ha sigut creat amb QGIS i s’ha exportat amb el pluggin qgis2web utilitzant la llibreria Leaflet.
El mapa ens mostra les principals discoteques, festivals i zones d’oci més importants d’Ibiza amb:
Marcadors interactius
Popups personalitzats
Informació sobre cada local
Agrupació de punts amb clusters

El mapa ha sigut incrustat a la web mitjançant un iframe.

Conclusions
Aquest projecte m’ha permés combinar els coneixements adquirits en aquesta assignatura de la creació d’una pàgina web, amb els coneixements que porto adquirint aquests anys de grau sobre la cartografia mitjançant QGIS, per a crear una pàgina funcional, visual i adaptada a diferents dispositius.
